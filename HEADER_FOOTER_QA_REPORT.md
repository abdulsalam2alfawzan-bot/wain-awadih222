# تقرير QA — Header وFooter الموحدان

## نطاق الفحص

تم فحص جميع ملفات HTML الموجودة في جذر المشروع، وعددها 12 صفحة.

## شروط القبول

لكل صفحة تم التحقق من:

1. وجود Header واحد فقط قبل `<main>`.
2. وجود شعار Header بمسار `assets/images/logo.png`.
3. وجود النص البديل `alt="وين أوديها"`.
4. ارتباط شعار Header بـ`index.html`.
5. وجود Footer واحد فقط بعد `<main>`.
6. وجود شعار Footer بالمواصفات نفسها.
7. وجود روابط: الرئيسية، متابعة الطلب، انضم كشريك، دخول الشريك، الخصوصية، الشروط.
8. وجود `aria-current="page"` لتعريف سياق الصفحة الحالية.
9. عدم بقاء عناصر `data-site-header` أو `data-site-footer` الفارغة.
10. سلامة الروابط والأصول عبر خادم HTTP محلي.
