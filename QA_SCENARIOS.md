# سيناريوهات QA

## رحلة العميل

1. مشكلة محرك واضحة → أربعة أسئلة → توقع وتخصص واستعجال وخطوة.
2. وصف غامض → أربعة أسئلة → عدم اختلاق سبب.
3. ضعف فرامل/رائحة وقود → استعجال خطر وتوجيه للسطحة.
4. قطع غيار مع VIN وقطعة وكالة.
5. قطع غيار دون VIN مع استمرار الرحلة وتنبيه المطابقة.
6. سطحة دون موقع → منع المتابعة.
7. سطحة دون وصف مكان → منع المتابعة.
8. صيانة دورية مع نوع وقود وخدمة.
9. منطقة/مدينة بلا Seed → حالة عدم تطابق صريحة.
10. تحديث الصفحة أثناء المسودة → استعادة البيانات والمرحلة.
11. إنشاء الطلب والإحالة قبل ظهور الشريك.
12. فتح WhatsApp ونسخ الرسالة.
13. إرسال رابط الطلب إلى جوال العميل عبر رسالة `wa.me` جاهزة.
14. طلب بديل ثانٍ وثالث واستبعاد السابق.
15. منع البديل الرابع.

## المتابعة والتقييم

16. تأكيد تلقي الخدمة دون تكلفة → بانتظار الفئة ولا رسم.
17. فئة أقل من 50 → رسم صفر.
18. فئة 50–أقل من 200 → رسم 5.
19. فئة 200–400 → رسم 10.
20. فئة أكثر من 400 → رسم 25.
21. إعادة تأكيد الفئة نفسها → لا رسم مكرر.
22. اختلاف الشريك والعميل على الفئة → تعليق للمراجعة.
23. منع تقييم إحالة غير منفذة.
24. تقييم موثق بعد الخدمة مع تعقيم التعليق.
25. إغلاق إداري بعد خمسة أيام دون حذف السجلات.

## الشريك والفوترة

26. دخول `WA-DEMO / 1234`.
27. تأكيد خدمة من لوحة الشريك وإدخال الفئة عند غياب إدخال العميل.
28. اعتراض على تلقي الخدمة.
29. اعتراض على تقييم.
30. اعتراض على فئة التكلفة.
31. تفعيل خصم مستمر وتعديله ثم إيقافه.
32. إصدار فاتورة يوم 1 برصيد أقل من 100 وترحيلها.
33. تحديث فاتورة الدورة عند بلوغ 100 دون تكرار العمليات.
34. منع اعتراض فاتورة بعد 15 يومًا.
35. تعليق الشريك عند تأخر فاتورة واجبة.
36. إعادة التفعيل بعد الدفع التجريبي.
37. إصدار وعرض وطباعة إيصال تجريبي.

## الانضمام

38. تسجيل ورشة مع سجل تجاري.
39. تسجيل محل قطع مع علامات مرتبة.
40. تسجيل مركز صيانة مع أنواع وقود.
41. تسجيل سطحة مع بطاقة تشغيل ومدن تغطية.
42. أيام عمل بالنقر وفترة ثانية اختيارية.
43. ترتيب التخصصات والعلامات صعودًا وهبوطًا.
44. منع الإرسال دون التعهد بالأمانة والموافقات.
45. متابعة طلب الانضمام بالرقم والجوال أو token.

## الجودة التقنية

46. فحص Syntax لكل JavaScript.
47. فحص الروابط المحلية والمعرفات المكررة.
48. فحص تسمية الحقول وأنواع الأزرار.
49. فحص RTL وViewport وCSP.
50. فحص CSS وreduced-motion.
51. تقديم جميع الصفحات والأصول عبر HTTP 200.
